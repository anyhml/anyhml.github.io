发表于 2017-08-28 | 分类于 Hadoop

前言

安装wget

简单版的centos7默认没有安装wget，需要自己手动安装
yum -y install wget

centos7关闭firewalld防火墙

基本使用

启动：systemctl start firewalld
查看状态：systemctl status firewalld
禁用：systemctl disable firewalld
禁用：systemctl stop firewalld

安装zookeeper集群搭建

如果没有安装wget，先安装wget。

  1. 首先创建目录，个人一般喜欢在/usr/local/下创建目录安装
